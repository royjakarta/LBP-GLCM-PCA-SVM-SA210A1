# Feature Cache Directory

This folder is used to store cached feature matrices generated by the main notebook.

The notebook extracts LBP and GLCM texture features from the metallographic images and saves the resulting feature matrices as compressed NumPy files (`.npz`). These files are generated automatically to reduce runtime when the same preprocessing and feature-extraction settings are reused.

Typical generated files include:

```text
features_lbp_glcm_training.npz
features_lbp_glcm_validation.npz
